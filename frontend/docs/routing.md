# Frontend Routing

## Routen
- `/login` -> LoginPage
- `/dashboard` -> DashboardPage
- `/score` -> ScoreEntryPage
- `/admin` -> AdminPage (captain)
- `/system` -> SystemAdminPage (admin)

## Guards
- Auth erforderlich ausser /login.
- Captain/Admin Guards fuer Rollenbereiche.

## Navigation
- Bottom-Nav zeigt nur erlaubte Bereiche.
